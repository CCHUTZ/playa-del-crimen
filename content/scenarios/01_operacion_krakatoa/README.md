# Escenario 01: Operación Krakatoa

**Inteligencia adversaria en conflictos corporativo-ambientales**

Este escenario te sumerge en una operación de inteligencia digital para desestabilizar la reputación de empresas turísticas acusadas de incumplir regulaciones ambientales. Aprenderás a investigar ecosistemas corporativos complejos, detectar vulnerabilidades reputacionales y construir narrativas basadas en datos públicos.
