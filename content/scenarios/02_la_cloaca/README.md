# Escenario 02: La Cloaca

**Inteligencia adversaria en corrupción política municipal**

Este escenario te sumerge en una operación de inteligencia digital para exponer sistemáticamente la corrupción municipal y generar rechazo social hacia un grupo político gobernante. Aprenderás a investigar estructuras de poder político local, rastrear el flujo de recursos públicos e identificar incumplimientos institucionales.
