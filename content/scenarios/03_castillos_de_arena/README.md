# Escenario 03: Castillos de Arena

**Inteligencia adversaria en fraude inmobiliario y rastreo de identidad**

Este escenario te sumerge en una operación de inteligencia digital para localizar, perfilar y reconstruir la red financiera y personal de un desarrollador inmobiliario fugitivo. Aprenderás a investigar identidades complejas, rastrear huellas digitales públicas y analizar patrones de comportamiento.
